Basic MediaElement.js demonstration plugin.

== Description ==

This plugin demonstrates some basic functionality of the MediaElement.js media player bundled with WordPress 3.6

== Installation ==

Install this just like any other plugin.  Then put a video somewhere on your site in order to see the results of the
options within this plugin.
